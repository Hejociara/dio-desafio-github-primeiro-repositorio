### Atalhos

Na Ide Eclipse

ctrl + 3    = abre uma janela onde você pode acrescentar o modo de perspectiva do git, debug, etc.

ctrl + n    = cria um novo projeto 

ctrl + espaço    =  dá opções sobre o que você digitou

f2 em cima da pasta = renomear 

/* = comentar

ctrl + 3 --> constru = cria uma contruction dos atributos criados 

Exemplo:

ctrl + d = apaga a linha

ctrl + 3 --> Tostring =  cria o to String 	

pressionar alt + seta p cima ou p baixo ao selecionar um bloco = Muda o bloco de lugar 

Colocar pontos separando as palavras ao se criar uma classe,  e add um nome em package, faz com que sejam separadas em pastas

ctrl + m = expande 

ctrl + Shift +F = organizar  

ctrl + alta + seta p baixo = copia um bloco de texto e cola abaixo

ctrl + 1 >>> assign statement variables . EX: digitar divisao (a,b);  ctrl + 1 >>> assing statement variables

ctrl + w  = fecha abas da IDE



Criar repositório do projeto para o Git dentro da IDE eclipse, clique na pasta do nosso projeto, ainda na IDE Eclipse, clique com o botão direito em team >>> share project >>>> create repository

Enviar projeto para o Git dentro da IDE eclipse, clicando na pasta do nosso projeto, ainda na IDE Eclipse, clique com o botão direito em team >>> share project >>>> commit >>> git >>

Commit (entregar) é a instrução que explicamos sobre o código que estamos subindo
