## Para empurrar o repositório para o GitHub

Abra o GitHub e o Gitbash no local da pasta ou pastas que desejas subir para a nuvem. Siga os passos:

Diigite os comandos: 

1 -  git init  --> enter

2 - git add * --> enter

3 - git commit -m "adicionarUmComentarioEntreAspas" --> enter

4 - git remote add origin AdicionarAChaveCriadaNoGitHub --> enter

5 - git push -u origin main

6 - atualize a página do GitHub no navegador e confira

## OU
